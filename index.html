School Programs

The 2017 Code fair will hold an event for high school students. This code event will engage school students in a fun and interactive learning experience. Students will develop new skills and also get important information on careers within the IT industry.

Hour of Code

This is an introductory crash course in coding. Students will take their very first steps in writing code and have fun along the way. This Hour of Code Workshop has been sponsored by The Discipline of Information Technology, Charles Darwin University in support of the Hour of Code initiative sponsored by Code.org.

Register your interest to be involved in this exciting workshop!


>More About Hour of Code

Students will work through a set of activities of varying degrees of difficulty; starting simple and progressing to slightly more challenging tasks.

The 2017 workshop will involve:

  • Introduction and overview
  • Interactive activities
  • Introduction to web design
  • Advanced coding topics
  • Resources to take home

CDU Code Fair History

The CDU IT Code Fair is an event that brings the IT community together. This is evident in the success of past events. Every year recognision is given to the achievements of students and feedback from all attendees is documented and taken into consideration for the preparation of future events.
